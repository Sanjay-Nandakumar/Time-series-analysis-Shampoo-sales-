About me

Hey there, I’m Sanjay.
This is a simple machine learning/deep learning - flask integrated web application created for learning purpose. The front end is entirely created using python flask scripts and some basic models in regression and classification are stored inside pickle files. These pickle files are loaded and executed whenever a predict function is called from the UI. I have also added some of my source code which I developed for learning ML/DL. The application is hosted in cloud via Amazon Web Services (AWS).
